# Vue-Sell-Cube
学习使用 Vue-cli 3.0 开发 饿了么 App 

下载：
```bash
git clone https://github.com/yangtao2o/vue-sell-cube.git
```

下载依赖：
```bash
npm install
```

运行：
```bash
npm run serve
```
